# Backend

## 개요
`backend`는 서비스 운영을 위한 서버 영역입니다. 회원 정보와 영상 데이터, 처리 상태를 관리하고 프론트엔드와 AI 파이프라인을 연결하는 역할을 합니다.

## 예정 기능 (간단 버전)
- 회원가입/로그인
- 사용자별 영상 업로드 및 목록 관리
- 처리 작업 상태 관리 (큐/잡 상태)
- 프론트엔드와 AI 파이프라인 연결을 위한 API 제공
